## Project Goal
- Use google map API with React.js

- Add extra information, wifi, plug, price range, etc. on every location.

- Manual location input

- User login, add, edit & delete route

- Input location from API (meetup, etc.)

## Tech Stack
- React

- Node.js

- google API

- ES6

- Auth

- SQLite3

- sass
